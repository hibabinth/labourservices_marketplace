# 🚀 Labrix (Labour Services Marketplace)

A Flutter-based mobile application that connects users with local service providers for various labour services.

---

## 📱 Features

- 🔍 Browse local service providers
- 📅 Book services (easy booking)
- 💳 Secure payments using Razorpay
- 👤 User authentication & profiles
- 📊 Real-time data with Supabase

---

## 🛠️ Tech Stack

- **Frontend:** Flutter
- **State Management:** Provider (MVVM)
- **Backend:** Supabase
- **Payments:** Razorpay

---

## 🏗️ Architecture

This project follows:

- MVVM (Model-View-ViewModel)
- Repository Pattern
